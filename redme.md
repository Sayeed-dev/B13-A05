2️⃣ What is the spread operator (...)?
It is like "unpacking" a bag. It takes all items out of an array or object and puts them into a new one.
it can use it to copy an array or combine two arrays into one very easily.

3️⃣ What is the difference between map(), filter(), and forEach()?
map() goes through an array and gives a new array with changes. filter() looks through an array and only keeps the items  want like finding only issues. forEach() just runs a function for every item but doesn't give  anything back .
